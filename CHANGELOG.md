## [1.0.0] - 2025-11-16
### Added
- Initial public release of the **WAW - Web Accessibility Widget**.
- Complete widget system with:
  - UI panel for all accessibility tools.
  - Configurable positioning (left or right).
  - Mobile and desktop detection.
  - Open/close interactions.
  - Persistent state management via LocalStorage.
  - Theme toggle (light / dark).
- Included accessibility features:
  - **Font size** adjustment.
  - **Line height** adjustment.
  - **Letter spacing** adjustment.
  - **inverted colors** mode.
  - **Grayscale** mode.
  - **Big cursor** mode.
  - **Reading guide line**.
  - **Reading mask**.
  - **Screen reader / Voice reading** support.
  - **Hide images** option.
  - **Highlight links**.
  - **Highlight headings**.
  - **Dyslexia-friendly font** mode.
  - **Mute all sounds**.
  - **Disable animations**.
- Integrated gesture support via **Hammer.js** for touch devices.
- Built-in CSS injection system (no external CSS required).
- Provided:
  - **ESM build**
  - **IIFE build**
  - External distributable CSS file for users who require direct import.
- Added full configuration system including:
  - Saving user preferences via LocalStorage.
  - Automatic loading on startup.
  - Auto-persisting changes during interaction.
- Project structure prepared for npm publishing with Rollup.

### Documentation
- Added full README with installation and usage instructions.
- Added examples for web integration.
- Added npm package metadata including keywords, repository info, and license.
